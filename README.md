# Chrome's New Minimalist Tab

Una extensión de Chrome que reemplaza la página de nueva pestaña por una vista limpia y minimalista, con buscador directo a Google y accesos rápidos a servicios de Google (Google, Gmail, Drive y YouTube).

![Vista previa](src/img/overview.png)

## Características

- **Nueva pestaña minimalista**
- **Buscador integrado** que envía consultas a Google
- **Accesos rápidos** a Google, Gmail, Drive y YouTube
- **Animaciones fluidas** al pasar el cursor sobre los accesos
- **Tema oscuro** con imagen de fondo
- **Sin permisos adicionales** (solo reemplaza la página de nueva pestaña)
